# Rails API for use with React-News Project
Created for use with [React-News project](https://github.com/LondonJim/React-News)

## Setup

install dependencies:

`bundle`

Testing:

`rails test`

Run locally (localhost:3000):

`rails server`

## Routes


#### Articles
GET /articles(.:format)
POST /articles(.:format)

#### Article
GET /articles/:id
PATCH /articles/:id
PUT /articles/:id
DELETE /articles/:id
